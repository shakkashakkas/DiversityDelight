# Diversity Delight
- A Farmer's Delight add-on that adds more crops to the game, and additional mechanics to make the farming/culinary experience more fun

### Side Note
Since the mod is very new into development and also being a complete amateur at coding and mod-making, there is a good chance you will encounter issues with this mod. Compatibility with other mods has not been considered as of yet, but if you have any suggestions or identify any issues, please send them to the GitHub issues page.
